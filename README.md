# 🚀 React Quiz App

A fully functional, interactive quiz application built with **React** and **TypeScript**. This app fetches trivia questions from an external API, manages user input, tracks scores, and provides immediate feedback.

---

## ✨ Key Features
* **Dynamic API Integration:** Fetches trivia questions in real-time from [Open Trivia DB](https://opentdb.com/).
* **TypeScript Excellence:** Strongly typed data structures to ensure code stability and bug-free logic.
* **Responsive UI:** A clean, user-friendly interface that looks great on all devices.
* **State Management:** Efficiently handles complex states (questions, answers, user selections, and scores).
* **HTML Entity Decoding:** Properly handles and parses encoded text from the API for a polished user experience.

---

## 🛠 Tech Stack
* **Frontend:** React, TypeScript, HTML5, CSS3.
* **Development Tools:** Git, npm, VS Code.
* **Data Source:** Open Trivia DB API.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. **Clone the repository:**
   ```bash
   git clone [(https://github.com/Youssef975825/QuizApp-TS-.git)]
   
   Navigate to the project folder:

  ```bash
cd [PROJECT_NAME]
```
Install dependencies:
  ```bash
npm install
```
Run the application:
  ```bash
npm start
```
